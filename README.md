# # Nike - Hungry For More (Conceptual Showcase)

> **Skill Showcase:** A high-motion, immersive conceptual product landing page for Nike, showcasing advanced scrolling interactions and 3D-like experience rendering.
>
> This project is a high-performance demonstration of creative front-end engineering, focused on storytelling through motion and interactive 3D elements.
>
> ---
>
> ## The Experience: High-Motion Storytelling
>
> The "Hungry For More" showcase pushes the boundaries of traditional web design by using a custom frame-by-frame rendering engine coupled with a high-fidelity animation timeline.
>
> ### Key Technical Features:
> - **Frame-by-Frame Canvas Engine**: A bespoke JS class that manages and renders 100+ high-quality product images to create a fluid, scroll-triggered 3D product reveal.
> - - **GSAP & ScrollTrigger Mastery**: Every transition, text reveal, and product transformation is choreographed into a single, cohesive scroll timeline for a professional "Apple-grade" experience.
>   - - **Responsive Canvas Hybrid**: A modular system that switches from interactive canvas rendering on desktop to optimized high-impact static backgrounds on mobile for guaranteed performance.
>     - - **Custom Smooth Scroll**: Integrated with **Lenis** to provide a premium, inertial scrolling feel that complements the motion-intensive UI.
>      
>       - ---
>
> ## The Tech Stack
>
> | Component | Technology | Rationale |
> |---|---|---|
> | **Core** | HTML5 / JavaScript | Maximum control over DOM and performance. |
> | **Logic** | Vanilla JS (ES6+) | Custom-built classes for canvas management. |
> | **Animation** | GSAP / ScrollTrigger | The industry standard for high-fidelity motion. |
> | **Scrolling** | Studio Freight Lenis | Provides the smooth, luxury feel required for high-motion sites. |
> | **Styling** | Vanilla CSS | Precision layout with custom typography (Antonio, Archivo). |
>
> ---
>
> ## Design Highlights
>
> - **Cinematic Visuals**: Uses full-bleed imagery and high-contrast typography to create an aggressive, athletic brand aesthetic.
> - - **Dynamic Content Ingestion**: A custom product-injection system allows for rapid scaling of product catalogs while maintaining consistent styling.
>   - - **Premium Navigation**: Glassmorphic, hide-on-scroll navigation that stays out of the way of the content while remaining accessible.
